CivicSuite / CivicZone

Zoning answers with citations, not guesses.

CivicZone is the planned parcel-aware zoning and land-use module for CivicSuite. It will help residents and planners find routine zoning information while escalating actual determinations to staff.

v0.1.0.dev0 runtime foundation

Current State

The package shell, root endpoint, health endpoint, documentation gates, canonical zoning schema, Alembic migration scaffold, and a sample parcel/zone lookup API exist. Live GIS ingestion, zoning Q&A, planner review queues, and public workflow screens are not implemented yet.

Product Promise

  • Show zone, overlays, mapped constraints, and citations.
  • Answer routine questions with source-backed informational language.
  • Refuse or escalate zoning-determination requests.
  • Keep staff-only precedent context out of resident-facing output.

Boundaries

CivicZone does not provide legal advice, does not make zoning determinations, and does not replace the planning department. It gives the 80% informational answer and routes judgment calls to humans.
